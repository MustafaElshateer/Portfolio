# Companys Portfolio

A personal portfolio website built with **Angular**, **TypeScript**, **HTML**, and **CSS**.

This portfolio presents my frontend development experience, skills, and selected projects, including Angular-based business applications, dashboards, authentication systems, and enterprise solutions.

## 🚀 About the Project

This project is a personal portfolio website created to showcase my work as a Front-End Developer specialized in Angular.

The portfolio includes:

- Personal introduction
- Technical skills
- Work experience
- Featured projects
- Contact information
- Responsive design for different screen sizes

## 🛠️ Technologies Used

- Angular
- TypeScript
- HTML5
- CSS3
- Angular CLI
- Responsive Web Design
